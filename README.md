# Movie-Service

- ✅ Java 

- ✅ Spring Boot

- ✅ Spring Security

- ✅ Hibernate

- ✅ Many to Many and One to One relationship

- ✅ Maven

- ✅ JPA

- ✅ PostgreSQL

- ✅ Postman


# About project

- Multi-user simple web service with Spring Boot that allows adding movies and series that can be watched by unique users.
